# 🐱펫퍼블릭

PETPUBLIC은 반려동물과 함께하는 일상을 기록하고, 동반 가능한 장소를 추천하는 플랫폼입니다. 함께하는 삶의 소중한 순간을 기록하고,
다른 반려동물을 사랑하는 이들과 경험을 나누며, 함께 즐길 수 있는 장소를 탐색할 수 있습니다!
